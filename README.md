# Semantic Mapper: XML to TTL

A browser-based tool for mapping XML structures to Turtle (TTL) vocabularies visually.

## Features
- **Visual Mapping:** Drag and drop fields from XML to TTL.
- **Dark Mode:** Professional IDE-like interface.
- **Privacy First:** Runs entirely in your browser; no data is sent to any server.
- **Export:** One-click JSON export of your mappings.
